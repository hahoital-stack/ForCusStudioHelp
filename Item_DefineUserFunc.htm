시나리오에서 특수한 목적으로 유저가 만든 라이브러리(linux so 형식 지원)를 동적으로 로드하여 사용 할 수 있도록 환경을 제공합니다.

PROPERTIES

 

User Function List

 

 

Up Button : 선택한 라이브러리를 위로 이동 합니다.

 

Down Button : 선택한 라이브러리를 아래로 이동 합니다.

X Button : 선택한 리스트를 잘라냅니다.(Ctrl+X)

C Button : 선택한 리스트의 내용을 복사 합니다.(Ctrl+C)

V Button : 복사한 리스트의 내용을 붙여넣기 합니다.(Ctrl+V)

E Button : 선택한 라이브러리의 정보를 수정 합니다. 리스트에서 라이브러리를 마우스 더블 클릭 해도 됩니다.(수정 창 팝업)

 

+ Button : 라이브러리를 추가합니다.(추가 창 팝업)

 

- Button : 선택한 라이브러리를 리스트에서 제거 합니다.

 

 

User Function : 라이브러리에 있는 사용할 함수를 지정 합니다.

User Library File : 라이브러리 파일을 지정 합니다.

User Library Type : 라이브러리 타입을 지정 합니다.(Ipron v5.3.0-b1 이상 지원)

Description : 지정한 함수를 설명 합니다.

 

『 라이브러리 파일의 등록은 운영관리의 설정을 참조 하시기 바랍니다 』
