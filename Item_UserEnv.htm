UserEnv 시작  다음  이전

ForCus 시스템 외에 특정한 환경 정보를 가져와 서비스에 적용하고 싶은 경우 운영관리(SWAT)에 사용자가 필요로 하는 값을 세팅하여 서비스 개발 시 환경정보를 가져와 시나리오 내에 사용 할 수 있도록 합니다.

 

 

ICON

 

 

 

PROPERTIES

 

Environment Data List

 

 

Name : 사용자의 환경 값을 저장 할 변수를 지정 합니다.

 

File : 환경 설정 파일을 지정 합니다.

 

Service : 환경 값을 가져올 서비스 아이디를 지정 합니다.

 

Value : 환경 값을 가져올 Key Value를 지정합니다.

 

Default : 환경 파일에서 값을 읽어 오지 못했을 경우 여기에 설정한 기본 값을 사용 합니다.

 

Write Check Box : Name 속성에 지정한 변수의 내용을 지정한 환경 파일에 기록 합니다.

 

Condition : 아이템 수행 여부 조건을 지정 합니다.

 

 

Tracking

 

 

트래킹을 처리합니다.

 

Properties

 

Check Tracking Check Box : Tracking을 사용 할 것인지를 체크합니다.

Tracking Point Flag Check Box : 특별한 Tracking Flag를 설정하여 SWAT에서 트래킹 시에 이 플래그가 설정된 항목만 트래킹 할 수 있습니다.

Encryption Value1 Check Box : Track Value1 속성 데이터에 대해서 암호화 처리를 합니다.

 

Track ID : 정의한 Track ID를 선택합니다.

- DefineTrack Item에 정의된 Track ID가 콤보박스 리스트로 제공됩니다.(Type이 없거나, UserEnv인 경우)

 

Track Name : Track ID를 선택하면 같이 정의된 이름을 출력합니다.(SWAT에서 이 이름으로 트랙킹 시에 사용합니다.)

 

Track Value1 : 트래킹 시에 사용할 값을 입력 합니다.

 

Track Value2 :트래킹 시에 사용할 값을 입력 합니다.(결과에 대한 Reason값)

 

 

RESULT
