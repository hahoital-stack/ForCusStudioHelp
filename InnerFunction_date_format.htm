DATE_STR date_format( DATE, DATE_FORMAT ) 시작  다음  이전

date_format

 

일자를 다양한 형태의 문자열로 변형해서 리턴 한다.

 

 

Parameter:

 

DATE [in] 기준 일자. YYYYMMDD
DATE_FORMAT [in] 날짜 문자열의 포맷을 나타내는 문자열로 일반적인 문자열과 예약어로 구성된다. 다음은 예약어 리스트이며, 예약어에 등록되지 않은 문자는 그대로 결과문자열에 남는다. 예약어 사용시 대소문자를 유념하여 사용해야 한다.
  %YYYY

: 4자리 연도
  %YY

: 2자리 연도
  %Y

: 연도
  %MM

: 2자리 월
  %M

: 월 (앞에 0 을 채우지 않는다)
  %DD

: 2자리 일
  %D

: 일 (앞에 0 을 채우지 않는다)

 

 

Return:

 

DATE_STR DATE 의 일자가 DATE_FORMAT 형태로 변경된 문자열

 

 

Remark:

 

ex)

date_format(‘20110317‘, ‘%Y/%M/%D’)

⇒ ‘2011/3/17’

 

date_format(‘20110317’, ‘[Time] %YYYY/%MM/%DD, %Y/%M/%D’)

⇒ ‘[Time] 2011/03/17, 2011/3/17’
