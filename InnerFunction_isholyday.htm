INT_NUMBER isholyday(DATE) 시작  다음  이전

isholyday

 

지정일 또는 오늘이 휴일인지 체크한다.

휴일 데이터는 운영관리(SWAT)의 휴일관리에 설정된 데이터를 기준으로 한다.

 

 

Parameter:

 

DATE [in] YYYYMMDD or MMDD or NULL

 

 

Return:

 

INT_NUMBER

1 :휴일

0 : 휴일이 아님

 

 

Remark:

 

ex)

# 평일

isholyday('20150520')

⇒ 0

 

# 크리스마스

isholyday('20151225')

⇒ 1

 

# 신정

isholyday('0101')

⇒ 1

 

# 오늘 날짜가 2015-05-20 일 경우(평일)

isholyday()

⇒ 0
