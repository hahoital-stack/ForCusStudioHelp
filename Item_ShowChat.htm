PROPERTIES

 

Chat Info

 

 

Chat Info

 

Text ID : 정의한 쳇 텍스트를 선택하거나 입력 합니다.

- DefineChatText Item에 정의된 TextID가 콤보박스 리스트로 제공되며 리스트에서 텍스트를 선택 할 수 있습니다.

- 텍스트가 들어 있는 변수를 사용 할 수 있습니다.

- 연산식을 통해 텍스트를 조합 할 수 있습니다. 단, 변수와 문자를 같이 연산 할 수는 없습니다.( 예 : gs_TrData.BI09ZZ.from.O_27_합계금액 * 1 )

- 텍스트 리스트에 여러개의 텍스트를 등록하여 전송 할 수 있습니다.

 

* Text ID 아래의 넓은 Editbox는 Type이 TextID일 경우에는 선택한 TextID에 해당하는 텍스트가 출력되고 Type이 Text일 경우에는 텍스트 를 편집할 수 있습니다.

 

Type : 전송할 텍스트 타입을 설정 합니다.

- TextID : Global의 DefineChatText 에 정의된 ID

- Text : 직접 전송할 텍스트 입력

 

Condition : 각 멘트의 재생 조건을 설정 합니다.

 

Up Button : 선택한 텍스트를 위로 이동 합니다.

 

Down Button : 선택한 텍스트를 아래로 이동 합니다.

Cut Button : 선택한 리스트를 잘라냅니다.(Ctrl+X)

Copy Button : 선택한 리스트의 내용을 복사 합니다.(Ctrl+C)

Paste Button : 복사한 리스트의 내용을 붙여넣기 합니다.(Ctrl+V)

Add Button : 지정한 텍스트를 추가 합니다.

 

Del Button : 선택한 텍스트를 삭제 합니다.

 

Update Button : 선택한 텍스트의 내용을 갱신 합니다.

 

Common

 

User Input Check Box : 사용자 입력을 받습니다.

Custom Keyboard Check Box : 가상 키보드를 전송합니다.

 

Keyboard Layout Info

 

Custom Keyboard 체크 박스를 체크하면 활성화 되며 가상 키보드를 설정합니다.

 

Layout ID Combo Box : DefineCustomKB Item에 정의된 Keyboard Layout을 선택합니다.

Update Text Button : 버튼의 텍스트를 수정합니다.(버튼 왼쪽 에디트 박스의 문자를 수정)

Custom Layout Check Box : DefineCustomKB에서 정의한 Layout을 사용하지 않고 버튼 레이아웃을 편집합니다.

Add Button : 버튼 레이아웃의 다음 라인에 버튼을 추가합니다.

Del Button : 버튼 레이아웃 에서 선택한 버튼을 제거 합니다.

 

 

 

RESULT

 

· 처리 결과 세션 변수

 

session.command.result : 처리 성공(_SLEE_SUCCESS_), 실패(_SLEE_FAILED_)

session.command.reason : 실패면 실패 오류코드(시스템 상수 참조)
