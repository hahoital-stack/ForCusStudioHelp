변수와 영역 시작  다음  이전

ForCus 서비스를 제공하기 위하여 필요한 다양한 변수를 ForCusStudo 에서 지원 합니다.

변수는 사용자가 정의 하는 변수 외에 세션변수가 존재 하는데 시스템에서 처리시 필요한 변수를 미리 정의해 놓은 것으로 전역으로 간주되며, 모든 시나리오 파일(.ssfx, .msfx, .usfx) 에서 사용가능 합니다.

 

 

◆ 변수 규칙

 

 

변수 타입

 

- ForCus 에서는 변수의 타입을 따로 지정하지 않으며, 변수에 들어가는 데이터의 타입에 따라 변수의 형이 결정됩니다.

 

예 : gName = 'abc' ⇒ 문자형 변수, gName = 123 ⇒ 정수형 변수

 

 

변수 구조 타입

 

- 변수를 선언하는 방법에 따라 일반, 배열 변수등으로 선언이 됩니다.

 

일반 변수

 

변수 선언 시 변수 이름을 단일 단어 형태로 선언 합니다.

 

예 : sName, g_nCount

 

 

배열 변수

 

변수 선언 시 변수 이름을 배열 형태로 선언 합니다. 배열의 첨자는 정의하는 대로만 참조 됩니다.

 

예 : sName[1..5] ⇒ sName 으로 배열 5 선언, sName[1] = 'a' ⇒ sName 1번 배열에 문자 'a' 입력

sName[11..20] ⇒ sName 으로 배열 10 선언, sName[11] = 'a' ⇒ sName 11번 배열에 문자 'a' 입력

abc[xxx] = 0 하면 abc로 한개짜리 배열이 생성되면서 0이 할당

 

 

구조체 변수

 

변수 선언 시 변수 이름을 구조체 형태로 선언 합니다.

 

예 : sClass.sName ⇒ sClass 변수 하위로 sName 변수를 선언

Global 변수로 sTable 선언 Local 에서 sTable.Name 선언 ⇒ Global 변수 sTable 의 하위로 Name 변수를 선언(즉, Local 에 선언 했지만 Global 로 간주)

sTable[1..5].Name ⇒ sTable 로 배열 5 선언하고 그하위로 Name을 각 선언

sTable.Name[1..5] ⇒ sTable 변수 하위로 Name 변수 배열 5 선언

 

 

변수 영역

 

- 변수는 지역변수와 전역변수로 구분되며 같은 변수가 지역과 전역에 있다면 전역변수가 사용이 됩니다.

 

※ ForCusStudio 에서는 전역과 지역에 같은 이름의 변수를 모두 선언 할 수 있지만 실제 SLEE 에서 변수가 생성되는 순서는 Global(Global.ssfx) 에 선언된 변수가 먼저 생성되고 Local(*.ssfx, *.msfx, *.usfx) 에 선언된 변수가 생성 되기 때문에 Local 변수 생성시 Global 로 생성한 변수가 있으면 변수 생성을 하지 않고 Global 로 선언된 변수를 사용 합니다.

 

전역 변수

 

Global.ssfx 시나리오에 정의한 변수를 말하며, 사용자 시나리오 에서는 사용할 수 없습니다.

 

지역 변수

 

Global.ssfx 를 제외한 모든 시나리오에서 정의한 변수를 말하며, 정의한 시나리오 내에서만 사용 가능 합니다.

 

 

변수 정의 방법

 

변수정의는 Var Item, Assign Item 에서 직접 선언이 가능하며, 그외 아이템에서는 아이템 이름이 변수가 되는 경우와 값을 할당 받는 경우인데 변수가 선언되지 않은 변수라면 선언하지 않아도 생성 됩니다.

 

Var Item

 

Global.ssfx 에서 선언된 Var Item 의 변수는 전역변수가 되고, 그 외에는 지역 변수 입니다.

 

 

Assign Item

 

Assign 에 변수를 선언하는 경우 전역, 지역 변수에 선언된 변수가 없으면 생성이 되며, 생성과 동시에 값을 할당 할 수 있습니다.
