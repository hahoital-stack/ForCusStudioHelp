insert

 

문자열 내에 특정 문자열을 삽입한다.

 

 

Parameter:

 

STRING [in] 입력 문자열
POS

[in] 삽입 위치(1-base)

유효한 값은 ‘1 ~ 입력 문자열 길이’사이이다.
INSERT_STR [in] 삽입할 문자열

 

 

Return:

 

OUTPUT_STR

문자열이 삽입된 결과 문자열

문자열 삽입이 실패하면, 입력 문자열을 리턴 한다.

 

 

Remark:

 

POS 입력 값이 숫자가 아니면, 입력문자열을 그대로 리턴 한다.

POS < 1 또는 POS 가 입력문자열 길이보다 크면, 입력문자열을 그대로 리턴 한다.

 

ex)

insert('Sunday Morning', 1, 'Refreshing')

⇒ 'Refreshing Sunday Morning'

 

insert('Sunday Morning', 8, 'Early ')

⇒ 'Sunday Early Morning'

 

insert('Sunday Morning', 0, 'Early')

⇒ 'Sunday Morning' : 삽입 위치가 0으로 삽입 실패
