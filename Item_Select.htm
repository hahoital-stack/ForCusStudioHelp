지정된 조건 식 또는 값의 비교할 구문이 다수일 경우 사용 합니다.

 

 

ICON

 

 

 

PROPERTIES

 

Select List

 

 

Conditional Expression : 비교할 조건 식이나 값을 지정 합니다.

조건값에는 관계 연산자를 사용할 수 있는데, 다음과 같은 연산자가 허용됩니다.

">", "<", "<=", ">=", "==", "!="

Condition 사용조건 :

- 관계연산자는 항상 값 왼쪽에 쓰여야 한다.

예) " >= 'TEST' " --> <if >= 'TEST' ...

- 관계연산자가 문자로 표시된 경우는 값으로 처리된다.

예) " '>= TEST' " --> <if == '>= TEST' ...

 

Type

Condition Radio Button : 비교할 조건문을 Start, End 항목을 통해서 지정 할 수 있습니다.

Default Radio Button : 기본 조건을 지정 할 수 있습니다.

 

Start : 조건문을 지정 합니다.(콤보 박스에 기본은 디지트 상수 값을 제공합니다.

 

End : Start와 함께 범위 조건문을 지정 할 수 있습니다.(숫자, 디지트인 경우)

 

Linked <next> Component : 조건문에 연결되는 아이템 이름을 지정 합니다. 아이템 이름은 타겟 아이템과 연결 시 자동으로 지정 됩니다.

 

Add Button : 비교 조건을 추가 합니다.

 

Del Button : 선택한 비교 조건 항목을 제거 합니다.

 

Update Button : 선택한 비교 조건 항목의 내용을 갱신 합니다.
