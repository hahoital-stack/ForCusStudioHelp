Menu 시작  다음  이전

ForCus Studio 메뉴의 기능 및 대표키, 단축키에 대하여 설명합니다.

 

 

Notes:

 

· E/D(Enable / Disable) : 파일의 변경 등 특정 동작일 때만 활성화 되는 메뉴

· S/H(Show/Hide) : 편집창이 있고 없고에 따라서 보이거나 숨겨지는 메뉴

 

 

File Menu

 

Menu Shortcut Hot Key Description E/D S/H
New Scenario Alt + F, N Ctrl + N 새 시나리오를 생성 합니다.    
New Project - - 새 프로젝트를 생성 합니다.    
Open Alt + F, O - 시나리오 파일 열기를 합니다.    
Close Alt + F, C - 열려 있는 시나리오 파일을 닫습니다.  
Close All Alt + F, E - 열려 있는 모든 시나리오 파일을 닫습니다.  
Create User Scenario - Ctrl + U 사용자 시나리오 파일을 생성 합니다.    
Edit User Scenario - - 등록된 사용자 시나리오 파일을 편집 합니다.    
Save Alt + F, S Ctrl + S 편집중 변경된 시나리오 파일을 저장 합니다.
Save As... Alt + F, A Ctrl + Shift + S 편집중인 시나리오 파일을 다른 이름으로 저장 합니다.  
Save All Alt + F, L - 편집중인 모든 변경된 시나리오 파일을 저장 합니다.
Open Project - - 프로젝트 파일 열기를 합니다.    
Save Project - - 프로젝트를 저장 합니다.  
Close Project - - 프로젝트를 닫습니다.  
Print... Alt + F, P Ctrl + P 현재 편집중인 시나리오를 인쇄 합니다.  
Print Preview Alt + F, V - 인쇄 미리보기를 합니다.(Scenario, Ment List, Packet List, Menu List)  
Print Setup Alt + F, R - 프린트를 설정 합니다.    
Recent Project - - 최근에 열었던 프로젝트 파일 리스트에서 프로젝트 열기를 합니다.    
Recent Scenario - - 최근에 열었던 시나리오 파일 리스트에서 열기를 합니다.    
Exit Alt + F, X - ForCusStudio를 종료 합니다.

Debug Menu

 

Menu Shortcut Hot Key Description E/D S/H
Start Debugging Alt + D, S F5 디버깅을 시작 합니다.  
Stop Debugging Alt + D, T Ctrl + F5 디버깅을 중단 합니다.  
Step Into - F11 디버깅 중 Subcall등 다른 시나리오 호출 아이템을 만났을 경우 호출 시나리오로 들어가 디버깅 합니다.  
Step Over - F10 디버깅 중 Subcall등 다른 시나리오 호출 아이템을 만났을 경우 시나리오로 들어가지 않고 호출 시나리오 디버깅을 합니다.  
Step Out - Shift + F11 현재 시나리오에서 더이상 Step별 디버깅을 하지 않고 남은 시나리오는 자동으로 디버깅을 진행 한 후 호출한 시나리오로 나갑니다.  
Run To Cursor - Ctrl + F10 커서가 있는 아이템 까지 디버깅을 진행한 후 멈춥니다.
