ch_availcnt

 

지정한 DNGroup의 가용 채널 수를 리턴한다.

 

 

Parameter:

 

STRING [in] 운영관리(SWAT)에서 설정한 DN 그룹명 문자열 데이터
STRING [in] 운영관리(SWAT)에서 설정한 Sub DN 그룹명 문자열 데이터, DN 그룹만 입력시 이값은 빈값으로('') 표시, DN 그룹명 없이 Sub DN 그룹명만 입력은 불가

 

 

Return:

 

RESULT

xx : 가용 채널 수

-1 : DN그룹 찾기 실패

 

 

Remark:

 

ex)

ch_availcnt('DNG_OUT1', '') : DM Group 만 입력 시

⇒ 29
ex)

ch_availcnt('DNG_OUT1', 'DNG_OUT1_SUB1') : Sub DN Group 입력 시

⇒ 29
