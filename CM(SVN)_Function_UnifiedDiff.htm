Unified Diff

 

- 선택한 파일의 현재 Revision과 바로 전 Revision 의 내용을 비교하여 변경된 부분을 보여 줍니다.

 

 

-(마이너스) 제거된 부분, +(플러스) 추가된 부분 으로 수정된 부분 입니다.(위 그림은 SCE 커맨드 아이템 위치가 변경된 경우)

 

- File 메뉴의 Find를 누르면 비교한 내용을 검색 할 수 있습니다.
