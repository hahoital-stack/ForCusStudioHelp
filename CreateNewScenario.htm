시나리오 생성하기 시작  다음  이전

새 시나리오 생성은 프로젝트를 생성하거나 시나리오 파일 하나를 생성 할 수 도 있습니다.

프로젝트를 생성하면 솔루션이 같이 생성되고 프로젝트는 솔루션에 종속되며 솔루션에는 여러개의 프로젝트를 추가할 수 있습니다.

 

 

· File 메뉴의 New Project를 선택하여 새로운 프로젝트를 생성 할 수 있습니다.

 

· File 메뉴의 New Service Scenario, New Menu Scenario를 선택하여 새로운 시나리오를 생성 할 수 있습니다. 또는 리본메뉴 Home> Open 에서도 가능합니다.

* 프로젝트 오픈 시에만 가능하며 프로젝트에 자동 추가 됩니다.

 

 

 

 

· 리본 메뉴의 Window > Window 에서 New Service Scenario, New Menu Scenario 를 선택하여 새로운 시나리오를 생성 할 수 있습니다.

* 프로젝트 오픈 시가 아니라도 가능하며 프로젝트에 자동으로 추가 되지 않습니다.
