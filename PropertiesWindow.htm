Properties Window 시작  다음  이전

Properties Window는 컴포넌트 아이템 클릭 시 아이템의 속성을 출력하는 윈도우 입니다..

 

 

Show / Hide

 

· 리본메뉴 View > Pane Window 카테고리 > Properties 체크

 

 

 

속성 메뉴

 

: 속성 트리 펼치기 / 닫기 - 속성 트리를 모두 펼치거나 모두 닫습니다.

 

: 속성 정렬해서 보기 - 속성을 알파벳 오름차순으로 정렬하여 표시 합니다.

 

: 속성 편집 잠금 / 해제 - 속성 편집을 못하게 잠그거나 해제 합니다.

 

: 속성 활성화 / 비활성화 - 눌려져 있어야 속성 내용이 표시 됩니다.
