token_iter

 

문자열에서 토큰을 차례대로 얻는다.

token_iter()함수 순차 호출에 토큰이 순서대로 리턴 된다.

 

 

Parameter:

 

STRING [in] 입력 문자열
DELIMITER

[in] 구분자 리스트

여러 개의 구분자를 입력할 수 있다.(공백도 구분자)

ex) ', ; * #'
POS_VAR

[in/out] 검색 시작 위치(1-base)

첫 번째 토큰을 구하기 위해서는 POS_VAR 변수 값에 ‘1’ 를 대입하지만, 이후 토큰부터는 검색 위치정보를 새롭게 입력할 필요는 없다. token_iter() 함수 내에서 자동으로 검색위치가 이동한다. 단, 자동으로 검색위치가 이동하려면 POS_VAR 에 변수가 입력 되어야 한다.( 예:'1' 을 변수에 넣고 변수를 대입)
RESULT_TOKEN [out] 결과 토큰

 

 

Return:

 

RESULT

성공 실패 여부.

성공이면 1, 실패이면 0을 리턴 한다.

 

 

Remark:

 

본 함수를 이용하여, 결과가 실패(=0)일 때까지 token_iter() 함수를 반복 호출하면 토큰을 처음부터 차례대로 얻을 수 있다. 첫 번째 호출 시에는 POS_VAR의 입력 변수에 ‘1’를 대입하여, 변수 자체를 입력하고, 이후 호출부터는 POS_VAR 변수만 내부 값의 변경 없이 재 입력한다. token_iter() 함수가 자동으로 POS_VAR 변수에 다음 검색 시작위치를 입력한다.

 

토큰과 구분자

 

문자열 '123;456;789;abc' 에서 구분자는 ';' 이며, 토큰은 '123', '456', '789', 'abc' 이다.

 

ex)

var sToken

var nPos = 2

token_iter('123;456;789;abc', ';', nPos, sToken)

⇒ sToken = '23'

 

token_iter('123;456;789;abc', ';', nPos, sToken)

⇒ sToken = '456'

 

token_iter('123;456;789;abc', ';', nPos, sToken)

⇒ sToken = '789'
