RouteSkill 시작  다음  이전

상담원 스킬 및 레벨에 따라서 호를 분배 하는 기능을 제공합니다.

 

 

ICON

 

 

 

PROPERTIES

 

Route Info

 

 

Skill ID : 상담원 스킬 ID를 입력 합니다.

 

Skill Level : 상담원 스킬 레벨을 입력 합니다.

 

Route Type : 호 분배 방법을 선택 합니다.(default : 최장대기)

 

Media Type : 미디어 타입을 선택 합니다.(음성)

 

BSR Routing : BSR 라우팅 적용 여부를 선택 합니다.

 

Route Priority : 라우팅 우선순위를 입력 합니다.

 

Common

 

멀티 그룹 라우팅의 공통 속성 입니다.

 

Priority : 그룹 라우팅의 라우팅 우선순위를 입력 합니다.

 

Timeout : 호 분배 시도 시간(초)을 입력 합니다.

 

BGM : 라우팅 하는 동안 BGM 을 Play 합니다.

 

Condition : 그룹 라우팅 아이템 수행 조건을 지정 합니다.

 

Route Info2 : 멀티 그룹 라우팅 2번 탭을 활성화 합니다.

 

Route Info3 : 멀티 그룹 라우팅 3번 탭을 활성화 합니다.

 

Route Info4 : 멀티 그룹 라우팅 4번 탭을 활성화 합니다.
