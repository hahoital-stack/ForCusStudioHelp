CdrCall 시작  다음  이전

CDR 을 호가 종료될 때 생성하지 않고 즉시 생성 합니다.

 

 

ICON

 

 

 

PROPERTIES

 

DB Procedure Info

 

 

DB Procedure Combo Box : DefineCDR에서 정의한 Stored Procedure를 선택합니다.

Procedure Desc : DefineCDR에서 정의한 Stored Procedure 설명 입니다.

Parameter : 선택한 Stored Procedure의 파라미터 리스트 입니다.

파라미터는 변수 이므로 각 변수에 값을 Assign 아이템을 이용해 할당 합니다.

Condition : 조건에 따라 CdrCall 아이템이 유효하거나 유효하지 않습니다.
