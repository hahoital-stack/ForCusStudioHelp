channelstate

 

입력한 채널(Leg)의 상태를 리턴한다.

 

 

Parameter:

 

NUMBER [in] SLEE 채널번호(-1 은 자기 채널)
STRING [in] MultiCall 문자열('First', 'Second'), ''(공백)은 'First' 로 처리

 

 

Return:

 

STRING

채널 상태 문자열

 

'----'' : NULL, 최초 값

 

[ Inbound ]

'Incom' : Incomming

'Ring' : Ringing

'ChatIncom' : Chat Incomming

'Hold' : Hold

 

[ MakeCall ]

'Make' : MakeInit

'Calling' : Calling

'Conn' : Connected

 

 

Remark:

 

ex)

channelstate(2, '')

⇒ 'Calling'
