Menu에서 Menu통계가 메뉴별로 처리되는데, 메뉴가 다르더라도 같은 의미의 내용을 하나의 통계로 처리하고 자 할 때 사용합니다.

예를 들면 메뉴 1번이 타행이체이고, 메뉴 2번이 당행이체면 같은 이체로 통계를 잡기 위해서 여기에 정의하는 Service Code를 지정하여 처리합니다.

PROPERTIES

 

Service Code List

 

 

List Control

 

Code : 서비스 코드 입니다.

Name : 서비스 코드에 의미를 부여하는 이름이며 SWAT에서 통계를 처리할 때 항목이름으로 사용합니다.

 

 

Up Button : 리스트에서 선택한 아이템을 위로 이동합니다.

 

Up Button : 리스트에서 선택한 아이템을 아래로 이동합니다.

 

Load Service Info Button : DefineMenu에서 설정되어 있는 Service Code를 가져와 리스팅 합니다.

 

Cut(X) Button : 선택한 리스트를 잘라냅니다.(Ctrl+X)

Copy(C) Button : 선택한 리스트의 내용을 복사 합니다.(Ctrl+C)

Paste(V) Button : 복사한 리스트의 내용을 붙여넣기 합니다.(Ctrl+V)

 

Add Button : Service Code를 추가합니다.

 

Del Button : Service Code를 리스트에서 제거 합니다.

 

Update Button : 선택한 Service정보를 갱신 합니다.
