Timeout

 

Use Timeout Check Box : 타임아웃 사용 체크 박스입니다. 타임아웃 시간을 지정합니다.

Timeout Chat : 타임아웃 시에 전송하는 쳇 텍스트를 지정합니다.

Description : Timeout Chat에 설정한 텍스트 내용입니다.

End Method : 타임아웃 시 처리 동작을 지정합니다.

 

Condition : 동작 조건을 지정합니다.
