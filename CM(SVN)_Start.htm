형상관리 시작

 

- 형상관리 켜기

 

형상관리를 사용하기 위해서는 기능을 켜야 합니다.

 

- SCE를 실행하고 리본 메뉴 Tools > Options > Configuration Management > Config 에서 Configuration Management On/Off 체크 박스를 체크하면 됩니다.

 

- 시나리오 오픈

 

형상관리 할 시나리오를(솔루션) 오픈합니다.

 

- 오픈한 시나리오가 형상관리중인 시나리오가 아니면 형상관리 기능 메뉴 중 Repository browser, Import, Settings 메뉴만 활성화 됩니다.(형상관리 중인 시나리오면 Import 메뉴는 비활성화 됩니다)

+ Repository browser : 원격 저장소 주소를 알고있으면 저장소를 탐색할 수 있습니다.

+ Import : 형상관리 할 시나리오를 최초 한번 원격 저장소에 업로드 합니다.(형상관리 여부 체크를 하기 때문에 버튼이 활성화 되는데 약간의 지연이 있을 수 있습니다.)

+ Settings : Import하기 전에 형상관리에서 제외할 파일 또는 폴더 등을 설정 합니다.

 

 

- Repository에 시나리오 등록(저장)

 

시나리오를 SVN 서버의 저장소로 Import 기능을 통해서 등록(저장) 하는 것이 시나리오 형상관리의 시작입니다.

 

- Import 전에 형상관리 하지 않을 파일 또는 폴더를 지우거나 제외 파일로 설정(Settings)할 수 있습니다.

- Import 할 때는 오픈된 프로젝트가 없어도 솔루션 디렉토리 하위의 모든 파일을 업로드 합니다.(제외로 설정한 파일 제외)

 

• 리본 메뉴 CM > Subversiond(SVN) > Settings 버튼 눌러서 제외되는 디렉토리, 파일 확인 및 설정

 

 

 

• 리본 메뉴 CM > Subversiond(SVN) > Import 버튼 누르기

 

 

Import 창이 팝업 되면 "Repository 그룹 Local : Default local reposi.." 에서 형상관리 할 디렉토리를 확인하고 Directory Confirm 체크 박스를 체크 합니다.

그리고 "Repository 그룹 URL of repository"에 원격 저장소 URL을 입력합니다.

마지막으로 Import Message를 입력후 OK버튼을 누르면 Import가 시작됩니다.(파일 개수가 많을 수록 Import시간이 오래 걸립니다.)

 

♣ 생성한 SVN 계정으로 완전 처음으로 Import등 SVN 서버를 접속해야 하는 경우에는 인증 창이 팝업 될 수 있는데 계정과 패스워드를 입력하면 됩니다.

 

 

 

※ 주의 : "Repository 그룹 Local : Default local reposi.." 에 기본적으로 설정되는 디렉토리는 SCE에서 오픈한 솔루션 파일(.sssx)이 있는 디렉토리 입니다.

- 만약 이 솔루션 파일이 여러개 프로젝트 중 특정 프로젝트 디렉토리 안에 있고 솔루션에 소속된 다른 프로젝트가 있는 경우 설정된 디렉토리를 그대로 사용하게 되면 솔루션 파일이 있는 프로젝트만 Import 되고 다른 프로젝트는 안되는 문제가 발생하므로 반드시 Import할 디렉토리를 확인하여 이 같은 경우라면 솔루션에 소속된 프로젝트 및 관련 디렉토리가 모두 포함될 수 있도록 디렉토리를 선택해 주시기 바랍니다.

 

- Import 대상 디렉토리 및 파일을 Add 하는 화면

 

- Add한 파일을 Commit 하는 화면

 

- CommitAdded된 항목중 실제 파일을 SVN서버로 전송하고 Import 완료된 화면(디렉토리가 제외 되므로 Added된 항목 보다 개수가 작음)

 

 

• Import 완료

Import 완료된 호면에서 OK버튼을 눌러 창이 닫히면 정상적인 경우라면 Solution View, File View 트리의 항목의 이미지가 SVN상태 이미지로 바뀌는 것을 확인 할 수 있습니다.

 

 

 

Import가 완료 되면 형상관리가 시작 됩니다.
