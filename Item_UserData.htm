PROPERTIES

 

User Info

 

 

Name : 사용자 데이터에서 읽은 값을 저장 할 변수를 지정 합니다.

 

Key : 사용자 데이터 값을 가져올 키를 지정 합니다.

 

Value : Update 시 에만 사용되며 업데이트 할 값을 입력 합니다.

 

Default : 사용자 데이터에서 값을 읽어 오지 못했을 경우 여기에 설정한 기본 값을 사용 합니다.

 

Updaet Box : Key에 지정한 사용자 데이터를 Value에 지정한 값으로 갱신 합니다.

 

Condition : 아이템 수행 여부 조건을 지정 합니다.
