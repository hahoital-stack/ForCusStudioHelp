token

 

문자열에서 지정 번째에 위치한 토큰을 리턴 한다.

 

 

Parameter:

 

STRING [in] 입력 문자열
DELIMITER

[in] 구분자 리스트

여러 개의 구분자를 입력할 수 있다.(공백도 구분자)

ex) ', ; * #'

TURN [in] 얻고자 하는 부분문자열 번째 (1-base)
RESULT_TOKEN [out] 결과 부분 문자열

 

 

Return:

 

RESULT

성공 실패 여부.

성공이면 1, 실패이면 0 을 리턴한다.

 

 

Remark:

 

토큰과 구분자

 

문자열 '123;456;789;abc' 에서 구분자는 ';' 이며, 토큰은 '123', '456', '789', 'abc' 이다.

 

ex)

var sToken

token('123;456;789;abc', ';', 1, sToken)

⇒ sToken = '123'

 

token('123;456;789;abc', ';', 4, sToken)

⇒ sToken = 'abc'
