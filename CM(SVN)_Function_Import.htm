Import

 

형상관리를 하기 위해서는 형상관리 할 시나리오를 선택하여 원격 저장소에 저장해야 합니다.

형상관리 시작 시 최초 한번 하는 기능으로 형상관리 대상 항목들을 원격 저장소로 전송합니다.

 

 

Local : 형상관리할 디렉토리를 설정합니다. 기본 값은 SCE 에서 오픈한 솔루션 폴더 입니다.

Directory Confirm Checkbox : Import 할 디렉토리를 확인 했다는 체크 박스로 체크해야 Import 가 처리 됩니다.

 

URL of repository : Import 할 원격 저장소 URL 주소 입니다.

 

- 원격 저장소 URL 설명

ex : https://100.100.108.188/svn/Repo/trunk

+ https://100.100.108.188/svn : SWAT 장비에 있는 SVN 주소

+ /Repo : Repository

+ /trunk : SVN 메인 트렁크로 Import하면 이 디렉토리 하위에 솔루션 디렉토리 하위의 디렉토리 파일들이 들어갑니다.(SVN은 branches, tags, trunk 3개로 구성)

 

Import message : Import 할 때 관련 정보를 입력 하면 어떤 내용으로 Import 되었는지 알 수 있습니다. 값을 입력해야 Import 가 처리 됩니다.

Recent message Button : 이전에 Import 성공한 입력 메시지를 최신 순으로 다시 선택할 수 있도록 제공 합니다.

 

Check the Global Ignore Pattern for directory and files to exclude from import Button : Import 하기 전 제외 파일을 확인하고 편집 할 수 있습니다.
