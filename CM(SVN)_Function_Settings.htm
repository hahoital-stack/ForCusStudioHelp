Settings

 

- 리본메뉴 CM > Subversiond(SVN) > Settings

 

 

디렉토리 하위의 항목을 Import, Commit, Add 등의 명령 처리 시 제외 할 디렉토리나 파일 패턴을 입력합니다.

 

ex) Backup : 백업 이라는 디렉토리 또는 파일 이름은 제외

*.zip : 파일 확장자가 zip 인 파일은 제외

 

패턴의 값은 기본 값이 있으며 사용자가 추가하면 레지스트리에 저장 됩니다.

개발 PC가 바뀌는 경우에는 사용자가 추가한 제외 패턴은 사라지니 백업 바랍니다.
