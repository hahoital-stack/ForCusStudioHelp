쳇팅 시에 문자(열)을 전송하지만 가상 키보드로 입력을 받을 때가 있는데 이때 가상 키보드를 화면에 출력하기 위한 키보드 레이아웃을 정의 하여 쳇 팅 시에 선택하여 사용한다.

PROPERTIES

 

 

Layout List

 

ID : Custom Keyboard Layout ID

Description : 설명

 

Button Layout

 

버튼을 추가하여 버튼 레이아웃을 구성합니다.

 

레이아웃에서 키를 누르면 같은 라인에 버튼을 추가 할 수 있습니다.

Update Text Button : 버튼의 텍스트를 수정합니다.(버튼 왼쪽 에디트 박스의 문자를 수정)

Add Button : 버튼 레이아웃의 다음 라인에 버튼을 추가합니다.

Del Button : 버튼 레이아웃 에서 선택한 버튼을 제거 합니다.

Add Layout Button : 편집 중인 버튼 레이아웃을 다른 이름으로 레이아웃 리스트에 추가합니다.

Update Layout Button : 편집 중인 버튼 레이아웃을 업데이트 합니다.

 

Up Button : 리스트에서 선택한 버튼 레이아웃을 위로 이동 합니다.

 

Down Button : 리스트에서 선택한 버튼 레이아웃을 아래로 이동 합니다.

 

X Button : 리스트에서 선택한 버튼 레이아웃을 잘라 냅니다.

 

C Button : 리스트에서 선택한 버튼 레이아웃을 복사 합니다.

 

V Button : 복사한 버튼 레이아웃을 리스트에 붙여 넣습니다.

 

E Button : 리스트에서 선택한 버튼 레이아웃의 정보를 편집 합니다.(리스트에서 더블클릭)

 

 

+ Button : 버튼 레이아웃을 추가 합니다.

 

- Button : 리스트에서 선택한 버튼 레이아웃을 제거 합니다.
