지정된 조건 식의 처리 결과가 참 또는 거짓에 대해 각 조건의 아이템으로 흐름이 이동 됩니다. 참인 경우 파란색 흐름 지시자, 거짓인 경우 주황색 흐름 지시자로 표시 되며, 참( O ), 거짓( X ) 으로 문자가 표시 됩니다.

 

 

ICON

 

 

 

PROPERTIES

 

Condition and Link

 

 

Conditional Expression : 조건 식이나 값을 지정 합니다.

 

True Case : 참인 경우 연결되는 아이템 이름을 지정 합니다. 아이템 이름은 타겟 아이템과 연결 시 자동으로 지정 됩니다.

 

False Case : 거짓인 경우 연결되는 아이템 이름을 지정 합니다. 아이템 이름은 타겟 아이템과 연결 시 자동으로 지정 됩니다.
