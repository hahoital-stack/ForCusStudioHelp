OUTPUT_STR replace( STRING, POS, LEN, REPLACE_STR ) 시작  다음  이전

replace

 

문자열의 일부분을 다른 문자열로 대체한 문자열을 리턴 한다.

 

 

Parameter:

 

STRING [in] 입력 문자열
POS [in] 변경 부분 시작 위치 (1-base)
LEN [in] 변경 부분 길이
REPLACE_STR [in] 대체할 문자열

 

 

Return:

 

OUTPUT_STR

결과문자열

치환 실패 시, 입력문자열을 리턴 한다.

 

 

Remark:

 

LEN, POS 입력 값이 숫자가 아니면, 입력문자열을 그대로 리턴 한다.

LEN < 0 또는 POS < 1 또는 POS 가 입력문자열 길이보다 크면, 입력문자열을 그대로 리턴 한다.

 

LEN 입력 값이 0 이면, 입력문자열에서 POS 위치부터 끝까지의 문자열을 대체할 문자열로 치환한 문자열을 리턴 한다. 입력문자열이 POS 위치부터 끝까지의 길이가 LEN 보다 작은 경우, 입력문자열에서 POS 위치부터 끝까지의 문자열을 대체할 문자열로 치환한 문자열을 리턴 한다.

 

REPLACE_STR 에 비어있는 문자열을 입력하면, POS위치부터 LEN길이까지 범위를 지운 문자열을 리턴 한다. erase()함수와 동일한 기능을 수행한다.

 

ex)

replace( 'Refreshing Sunday morning', 12, 6, 'Monday' )

⇒ 'Refreshing Monday Morning'

 

replace( 'Refreshing Sunday morning', 12, 0, 'Monday')

⇒ 'Refreshing Monday' : 변경 부분 길이가 0인 경우 Pos부터 끝까지 변경 대상

 

replace( 'Refreshing Sunday morning', 12, 20, 'Monday')

⇒ 'Refreshing Monday' : 변경 부분 길이가 Pos부터 문자열 끝까지 길이보다 클 경우
