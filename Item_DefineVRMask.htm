DefineVRMask 시작  다음  이전

메뉴트리 음성인식 인식어 리스트를 지정 합니다. 메뉴트리 구성에서 기본적으로 하위메뉴의 인식어가 기본 인식어(리스트)가 되고, 여기서 정의하는 인식어 리스트가 추가 된다.

 

 

ICON

 

 

 

PROPERTIES

 

VR Mask

 

 

Caption Ment

 

Mask ID : Mask ID 를 정의 합니다. 이값은 DefineMenu의 VR Option 및 MenuChange에 사용 됩니다.

Set Common Check Box : 공통 Mask ID로 설정 합니다.

Mask Description : 정의한 Mask ID를 설명합니다.

Common Mask ID : 공통으로 정의된 Mask ID 를 지정 합니다.

 

Caption Ment

Mask String : 인식어를 입력합니다.

Mask Value : 인식어와 연결되어 처리될 값을 지정 합니다.(메뉴 아이디)

Description : 지정한 메뉴 이름이 표시 됩니다.

Delete ALL Button : Mask 그룹의 리스트를 모두 삭제 합니다.

Add Button : 입력한 인식어 및 처리 값을 추가 합니다.

Del Button : 입력한 인식어 및 처리 리스트에서 선택한 내용을 삭제합니다.

Update Button : 입력한 인식어 및 처리 리스트에서 선택한 내용을 갱신합니다.

New Button : 선택되어 표시된 모든 Properties를 초기화 합니다.

Add Button : 정의한 Mask ID 및 포함됨 Mask 를 리스트에 추가 합니다.

Del Button : 리스트에서 선택한 Mask ID를 삭제 합니다.

Update Button : 리스트에서 선택한 Mask ID의 정보를 갱신 합니다.

 

RESULT

 

None.
