Assign 시작  다음  이전

변수에 값이나 연산식을 통하여 값을 설정합니다. 설정 변수의 이름이 지역변수와 전역변수에 동일하게 존재하는 경우 지역변수가 우선이 됩니다.

정의 되어 있지 않은 변수에 값을 할당 하는 경우는 변수를 지역변수로 새롭게 생성하고, 값을 설정 합니다.

내장 함수를 호출하여 사용할 수 있으며, 그 결과 또한 사용이 가능합니다.

외부 함수 호출은 허용 하지 않습니다.

PROPERTIES

 

Assign List

 

 

List Box : 여러개의 변수에 값을 할당 할 수 있으며 리스트로 구성 됩니다.

 

· No. 리스트에 추가한 변수 또는 연산식의 카운트를 표시 합니다.

 

Up Button : 선택한 변수를 위로 이동 합니다.

 

Down Button : 선택한 변수를 아래로 이동 합니다.

 

X Button : 선택한 변수를 잘라 냅니다.(Ctrl+X)

 

C Button : 선택한 변수를 복사 합니다.(Ctrl+C)

 

V Button : 복사한 변수를 붙여넣기 합니다.(Ctrl+V)

 

E Button : 선택한 변수를 수정 합니다.

 

+ Button : 변수를 추가 합니다.

 

- Button : 선택한 변수를 삭제 합니다.

 

 

Variable Name : 변수명 지정

Expression : 변수, 상수, 내장함수, 연산자를 사용 할 수 있으며, 변수의 설정 값으로 사용 됩니다.

Condition : 변수에 값을 설정할 때 설정 여부의 조건을 지정 합니다.
