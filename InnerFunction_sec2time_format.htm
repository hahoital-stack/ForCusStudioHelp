sec2time_format

 

초시간과 타임포멧을 입력받아 타임포멧 형태의 문자열을 리턴한다.

 

 

Parameter:

 

SECOND [in] 초시간
TIME_FORMAT [in] 시각 문자열의 포맷을 나타내는 문자열로 일반적인 문자열과 예약어로 구성된다. 다음은 예약어 리스트이며, 예약어에 등록되지 않은 문자는 그대로 결과문자열에 남는다. 예약어 사용시 대소문자를 유념하여 사용해야 한다.
  %hh

: 2자리 시간
  %h

: 시간 (앞에 0 을 채우지 않는다)
  %mm

: 2자리 분
  %m

: 분 (앞에 0 을 채우지 않는다)
  %ss

: 2자리 초
  %s

: 초 (앞에 0 을 채우지 않는다)

 

 

Return:

 

TIME_STR TIME_FORMAT 형태의 결과 문자열

 

 

Remark:

 

ex)

sec2time_format(340, ‘%hh:%mm:%ss’)

⇒ ‘00:05:40’

 

sec2time_format(340, ‘%h시간 %m분 %s초')

⇒ ‘0시간 5분 40초 ’
