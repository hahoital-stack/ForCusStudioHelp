Repository browser 기능

 

 

: 화면 갱신 버튼 입니다.(F5 키, Popup Menu > Refresh)

URL : 현재 선택되어 있는 저장소 주소 경로 입니다.

: 현재 경로에서 상위로 이동 버튼 입니다.

Details : 항목 들을 리스팅 할 레이아웃을 선택 합니다.

 

Tree : 저장소에 업로드 된 형상관리 시나리오 트리 입니다.

- 위 그림에서 Genesys, SolutionTestSWP, Solution_Test 는 Working copy의 솔루션 디렉토리와 이름이 같습니다.

- 각 솔루션 디렉토리 하위에 branches, tags, trunk 기본 디렉토리가 있습니다.

- trunk 디렉토리는 메인 개발 디렉토리 이며 시나리오 Import시에 Repository 주소 끝에 /trunk 로 Import 하며 이 디렉토리 하위에는 로컬의 형상관리 디렉토리 Root 하위에 있는 디렉토리와 파일들이 있습니다.

- Solution_Test > trunk 디렉토리 하위의 CopyProject, IRTEST, IRTEST_DUP, 블록 시험 시나리오-IR_MAIN_NMS 디렉토리는 프로젝트 이며, 나머지 디렉토리는 공통 디렉토리 입니다.

 

List : 선택한 저장소 경로 하위의 디렉토리 및 파일을 리스팅 합니다.

- File : 디렉토리와 파일 이름 입니다.

- Extension : 파일인 경우 파일 확장자 입니다.

- Revision : 디렉토리 또는 파일이 Commit 시에 부여된 관리 번호 입니다.

- Author : Commit 한 SVN 계정 입니다.

- Size : 파일인 경우 파일 사이즈 입니다.

- Date : Commit 된 날자/시간 입니다.

- Lock : 파일인 경우 파일에 락이 걸려 있는지 표시 합니다.

- Lock comment : 파일에 락을 걸 때 입력한 메시지 입니다.

 

Popup Menu : 디렉토리와 파일항목에서 마우스 오른 버튼을 누르면 팝업됩니다.
