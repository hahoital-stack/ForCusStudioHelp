erase

 

입력 문자열에서 일 부분을 제거한 문자열을 리턴 한다.

 

 

Parameter:

 

STRING [in] 입력 문자열
POS [in] 제거할 시작 위치 (1-base)
LEN [in] 제거할 길이

 

 

Return:

 

OUTPUT_STR

일 부분이 제거된 결과 문자열

제거 실패 시 에는, 입력 문자열 전체를 리턴 한다.

 

 

Remark:

 

POS, LEN 입력 값이 숫자가 아니면, 입력 문자열을 그대로 리턴 한다.

LEN < 0 또는 POS < 1 또는 POS 가 입력문자열 길이보다 크면, 입력문자열을 그대로 리턴 한다.

 

만약 LEN 입력 값이 0 이면, 입력문자열에서 POS 위치부터 끝까지 제거된 문자열을 리턴 한다.

만약, 입력문자열이 POS 위치부터 끝까지의 길이가 LEN 보다 작다면, POS 위치로부터 문자열 끝까지 제거된 문자열을 리턴 한다.

 

ex)

erase( 'Refreshing Sunday morning', 12, 7)

⇒ 'Refreshing Morning'

 

replace( 'Refreshing Sunday morning', 12, 0)

⇒ 'Refreshing ' : 제거할 길이가 0인 경우 Pos부터 끝까지 제거

 

replace( 'Refreshing Sunday morning', 12, 20)

⇒ 'Refreshing ' : 제거할 길이가 Pos부터 문자열 끝까지 길이보다 클 경우
