VisualArs 시작  다음  이전

Callgete사의 푸시 솔루션인 VisualArs에 푸시 메시지를 전송하기 위해 푸시 메시지를 편집하는 아이템 입니다.

 

VisualArs는 IVR에서 Play되는 서비스메뉴를 스마트폰 화면에 출력하여 눈으로 정확히 보고 모든 메뉴를 듣지 않고서도 메뉴를 선택할 수 있습니다.

 

* 이 아이템은 ForCus 4.3 이상 버전에서만 사용 가능 합니다.

 

 

ICON

 

 

 

PROPERTIES

 

Visual ARS Message Edit

 

 

 

Name : 아이템 이름을 지정 합니다.(편집한 메시지 내용이 들어있는 변수)

 

Select Ment : 멘트를 선택하면 멘트 Description이 Message Edit Box 현재 커서 위치에 붙여 넣어 집니다.

 

• 선택 멘트 리스트

 

Using Expression Check Box : Message Edit Box의 내용을 연산식으로 처리 합니다.

 

• Message Edit Box

- Edit Box에서 선택한 문자열 위치 에서 Enter키를 누르면 선택한 문자열 다음 문자열들은 다음줄에 표시 됩니다. 하지만 빌드 시 줄바꿈 문자인 "\r\n"은 제거 됩니다.

ex) 멀티멘트 시험은 1번,     ===>      멀티멘트 시험은 1번, 없는 파일 안내 2번

     없는 파일 안내 2번

- 메시지 편집 시 "\r\n"이 아닌 줄 바꿈 문자가 있다면 입력 하면 됩니다.

 

Preview Button : VisualArs Push 미리 보기

 

Del Button : Select Ment에서 멘트를 선택하면 멘트 리스트에 선택한 Ment와 Description이 추가되는데, 멘트 리스트에서 선택한 멘트를 제거한다.

 

Insert Message >> Button : 멘트 리스트에서 선택한 멘트의 Description을 Message Edit Box의 현재 커서 위치에 붙여 넣습니다.(선택 멘트 리스트에서 멘트를 더블 클릭해도 됩니다)

 

Condition : 아이템 수행 조건을 지정 합니다.
