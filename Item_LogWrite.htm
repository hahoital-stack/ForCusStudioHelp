LogWrite 시작  다음  이전

사용자가 원하는 로그를 별도의 로그 파일에 기록 할 수 있습니다.

로그에 대한 정의는 DefineLog Item 에 정의 됩니다.

 

 

ICON

 

 

 

PROPERTIES

 

Log Write

 

 

Slee Log Check Box : 로그를 LOG ID에 지정된 곳이 아닌 SLEE 로그에 출력 합니다.

 

Log ID : DefineLog 에 정의한 Log ID를 지정 합니다.

 

Log Path : 지정한 Log ID 에 정의된 Log Path 를 표시 합니다.

 

Expression : 로그 데이터를 입력 합니다.(시작과 끝에 문자열 표시 작은 따옴표 ' ' ))

 

Format Ex : Java 형식의 문자열 데이터를 지원합니다.(시작과 끝에 문자열 표시 없음, 빌드 시에 ForCus 문자열 형식으로 변환됨)

 

Condition : 아이템 수행 여부 조건을 지정 합니다.

 

 

RESULT

 

None.
