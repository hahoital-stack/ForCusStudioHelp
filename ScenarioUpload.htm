시나리오 업로드 시작  다음  이전

컴파일된 시나리오 파일이며 서비스 단위인 "*.SXML" 을 지정된 서버로 전송 합니다.

전송 정보의 설정은 Tools>Options 의 Upload Tab 에서 설정 할 수 있습니다.

 

 

 

Project>Scenario Upload 메뉴를 선택하면 위와 같은 Upload Dialog 가 팝업 됩니다.

 

Upload Information

 

- 설정한 FTP 전송 정보가 표시 됩니다.

 

Upload Button : 업로드를 시작합니다.(백업이 완료되면 OK 버튼으로 바뀝니다)

 

- Upload 시에는 파일 이름을 사용자가 지정한 Scenario Version 과 Build Number 를 조합하여 바꿔서 전송하는데 시나리오파일의 버전을 관리하기 위함 입니다.

- Scenario Version 은 Project>Settings 에서 지정 할 수 있으며 Build Number 는 빌드한 "*_ALL.SXML" 파일이 지워 지지않으면 계속 증가를 하며, 지워지면 0부터 다시 시작합니다.

- Scenario Version 은 지정하지 않아도 되는데, 이때는 Build Number 만 전송 파일 이름으로 조합 됩니다.

전송파일 조합 순서 : “ *_ALL.SXML ” + Scenario Version + Build Number
