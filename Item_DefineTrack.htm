Tracking에 사용할 Track ID를 정의 합니다.

 

 

ICON

 

 

 

PROPERTIES

 

Tracking ID List

 

 

List Control

 

ID : Track ID 입니다.

Type : 타입은 GetDigit, Play, Packet, Cti, Query와 타입이 없는 타입이 있는데 각 타입은 해당되는 아이템에서 Track ID을 입력 할 때 아이템에 해당 하는 Track ID만 리스팅 하기 위해서 사용 됩니다.(타입이 없는 타입은 모든 아이템의 Track ID에 리스팅 되며, Tracking Item의 Track ID에는 모든 속성의 타입이 모두 리스팅 됩니다.)

Name : Track ID에 의미를 부여하며, 트래킹 시에 SWAT에서 이 이름을 활용합니다.

 

Up Button : 리스트에서 선택한 아이템을 위로 이동합니다.

 

Up Button : 리스트에서 선택한 아이템을 아래로 이동합니다.

 

Import : Excel로 작성된 Track ID를 읽어 들입니다.

 

Export : Track ID를 Excel파일로 생성합니다.

 

Cut(X) Button : 선택한 리스트를 잘라냅니다.(Ctrl+X)

Copy(C) Button : 선택한 리스트의 내용을 복사 합니다.(Ctrl+C)

Paste(V) Button : 복사한 리스트의 내용을 붙여넣기 합니다.(Ctrl+V)

 

Add Button : Track ID를 추가합니다.

 

Del Button : Track ID를 리스트에서 제거 합니다.

 

Update Button : 선택한 Track 정보를 갱신 합니다.
