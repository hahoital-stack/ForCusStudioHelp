PROPERTIES

 

Cti Info

 

 

Tracking Point Flag Check Box : 특별한 Tracking Flag를 설정하여 SWAT에서 트래킹 시에 이 플래그가 설정된 항목만 트래킹 할 수 있습니다.

Encryption Value1 Check Box : Track Value1 속성 데이터에 대해서 암호화 처리를 합니다.

 

ID : 정의한 Track ID를 선택합니다.

- DefineTrack Item에 정의된 Track ID가 콤보박스 리스트로 제공됩니다.(모든 타입)

 

Name : Track ID를 선택하면 같이 정의된 이름을 출력합니다.(SWAT에서 이 이름으로 트랙킹 시에 사용합니다.)

 

Value : 트래킹 시에 사용할 값을 입력 합니다.

 

Tracking Point Flag Check Box : 특별한 Tracking Flag를 설정하여 SWAT에서 트래킹 시에 이 플래그가 설정된 항목만 트래킹한다.

 

Condition : 아이템 수행 조건을 지정합니다.
