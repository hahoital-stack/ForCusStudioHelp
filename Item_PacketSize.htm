PacketSize 시작  다음  이전

Send 전문 사이즈를 구할 수 있는 아이템 입니다.

 

전문 정의는 Global에 정의 되어 있으므로 PacketSize 아이템을 시나리오 어디에서도 적용 가능 하지만 실제 사이즈를 정확하게 얻기 위해서는 PacketCall 아이템 전에 호출 되어야 합니다.

 

* 이 아이템은 ForCus 4.3 이상 버전에서만 사용 가능 합니다.

 

 

ICON

 

 

 

PROPERTIES

 

Packet Info

 

 

Name : 아이템 이름을 입력합니다.(결과 시스템 변수를 참조할 때 사용 합니다), PacketCall Item 의 이름을 입력 합니다.

 

Packet ID : 전문 사이즈를 구할 정의된 Packet을 지정 합니다.

 

Open Packet Item Button : Packet ID에 지정된 정의된 DefinePacket Item으로 이동 합니다.

 

Condition : 아이템 수행 조건을 지정 합니다.

 

 

RESULT

 

· 아이템 속성 변수(@name 은 Name 항목에서 지정한 이름)

 

@name.sendsize : 전송할 패킷 데이터 사이즈(반복 부 포함)

@name.sendpartsize : 전송할 패킷 데이터 사이즈(반복 부 제외)

@name.sendrepeatsize[XX] : 반복 부별 정의된 보내는 전문 사이즈(반복 부는 9개 까지 지원), XX(repeat 카운트 인덱스, 1 base)

@name.repeatcount : 전송할 패킷에 포함된 반복 부 개수
