replace_chars

 

문자열 내에 특정 문자들을 다른 문자로 대체한 문자열을 리턴 한다.

 

 

Parameter:

 

STRING [in] 입력 문자열
CHARS [in] 교체 대상 문자 리스트(대소 문자 구분)
REPLACE_CHAR [in] 대체문자

 

 

Return:

 

OUTPUT_STR

결과 문자열

치환 실패 시, 입력문자열을 리턴 한다.

 

 

Remark:

 

교체 대상 문자 리스트(CHARS)에 대체문자(REPLACE_CHAR)가 포함되어 있으면 치환작업을 수행하지 않고, 입력 문자열을 그대로 리턴 한다. 대체문자(REPLACE_CHAR)는 하나의 문자이어야 하며, 만약 문자열이 입력된 경우 첫 번째 문자만을 대체문자로 간주한다.

 

REPLACE_CHAR 에 비어있는 문자를 입력하면, 교체 대상 문자를 지운 문자열을 리턴 한다. erase_chars() 함수와 동일한 기능을 수행한다.

 

ex)

replace_chars( 'Refreshing Sunday morning', 'Sm', 'Q' )

⇒ 'Refreshing Qonday Qorning'

 

replace_chars( 'Refreshing Sunday morning', 'Sm', 'm' )

⇒ 'Refreshing Sunday morning' : 대체문자가 교체대상 문자에 포함되어 있어 치환 실패

 

replace_chars( 'Refreshing Sunday morning', 'Sm', 'Zero' )

⇒ 'Refreshing Zunday Zorning' : 대체문자가 문자가 아닌 문자열이 입력되어 첫 번째 문자로만 대체
