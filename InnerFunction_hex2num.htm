hex2num

 

Hexa String 을 숫자로 변환

 

단, 내부적으로 32bit 범위의 데이터를 사용하기 때문에 Hexa String 기준 0000 0000 ~ FFFF FFFF 까지의 범위만 변환 된다.

그러나 부호를 표시 하기 위한 맨 앞 1bit 를 부호 표기에 사용하므로 정수 범위는 –2,147,483,648 ~ 2,147,483,647 값으로 반환된다.

 

∗ ForCus v5.2.0-b3 이상 지원

 

 

Parameter:

 

STRING [in] 입력 Hexa String

 

 

Return:

 

NUMBER 결과 숫자

 

 

Remark:

 

ex)

hex2num( 'A' )

⇒ 10

 

hex2num( 'F' )

⇒ 15

 

hex2num( 'AB' )

⇒ 171

 

hex2num( 'FFFFFFFF' )

⇒ -1
