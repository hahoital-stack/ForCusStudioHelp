hex2str

 

Hexa Code 화 된 문자열을 일반 문자열로 반환

 

∗ ForCus v5.2.0-b2 이상 지원

 

 

Parameter:

 

STRING [in] 입력 Hexa String

 

 

Return:

 

OUTPUT_STR 결과 문자열

 

 

Remark:

 

ex)

hex2str( '54657374' )

⇒ 'Test'
