SUB_STR rsubstr( STRING, POS, LEN ) 시작  다음  이전

rsubstr

 

문자열중 일부 문자열을 리턴 한다.

 

 

Parameter:

 

STRING [in] 문자열 데이터
POS [in] 부분 문자열의 시작 위치(1-base)
LEN [in] 부분 문자열의 길이

 

 

Return:

 

SUB_STR

부분 문자열

실패 시 NULL(POS, LEN 이 유효하지 않은 값, 기타)

 

 

Remark:

 

LEN, POS 입력 값이 숫자가 아니면, 문자열을 리턴 하지 않는다.

LEN < 0 또는 POS < 1 또는 POS 가 입력문자열 길이보다 크면, 문자열을 리턴 하지 않는다.

 

LEN 입력 값이 0 이면, 입력문자열에서 POS 위치부터 끝까지의 문자열을 리턴 한다.

입력문자열이 POS 위치부터 끝까지의 길이가 LEN 보다 작은 경우, POS 위치부터 끝까지의 문자열을 리턴 한다.

 

ex)

rsubstr( 'Refreshing Sunday morning', 12, 6)

⇒ 'hing S'
