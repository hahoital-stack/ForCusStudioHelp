Show Log

 

- 디렉토리 또는 파일이 Commit 된 Revision 번호를 리스팅 하며 각 Revision 별 로 상세한 정보를 볼 수 있습니다.

 

 

Filter : 입력한 값을 리스트에서 Author, Date, Message 항목의 값과 비교하여 같은 문자가 있는 Revison만 리스팅 합니다.

Author Checkbox : Author를 필터링 항목으로 체크, 체크해제 합니다.

Date Checkbox : Date를 필터링 항목으로 체크, 체크해제 합니다.

Message Checkbox : Message를 필터링 항목으로 체크, 체크해제 합니다.

 

Lookup period revision Checkbox : 지정한 Revision 번호 사이의 Revision을 조회 합니다.

Lookup period date Checkbox : 지정한 날자에 해당 하는 Revision을 조회 합니다.

 

Revision List : 지정한 날자에 해당 하는 Revision을 조회 합니다.

Message Box : Revision List 에서 선택한 Revision 이 Commit 될 때 입력한 메시지를 표시 합니다.

Item List : Revision List 에서 선택한 Revision 에 같이 Commit 된 항목 리스트를 리스팅 합니다.

 

Retrieve stop on copy/rename Checkbox : 복사, 이름 변경이 일어난 지점 까지만 로그를 수집합니다.(해당 branches의 변경 사항 만 표시할 때 유용합니다.

Retrieve merged revisions Checkbox : Merged Revision 로그를 수집합니다.

Refresh Button : 리스팅 화면을 갱신합니다.

 

Popup Menu : Item List의 항목에서 마우스 오른 버튼을 누르면 팝업됩니다.

Unified diff : 선택한 파일의 현재 Revision과 바로 전 Revision의 내용을 비교.

Blame : 선택한 파일의 각 라인 별로 Commit한 작성자, Revision, 수정 날자등을 보여줌.
