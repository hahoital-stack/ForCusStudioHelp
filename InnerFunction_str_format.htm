str_format

 

다양한 형태의 문자열을 생성한다.

 

* str_format 함수는 v5.0, v5.1 SIP 버전 전용(v5.1 NMS 버전은 지원 않함)

 

Parameter:

 

FORMAT

[in] 결과 문자열의 형태를 정의하는 문자열. 일반적인 문자열과 예약어로 구성 되며, 일반적인 문자열은 그대로 결과 문자열을 이루고, 예약어는 다음의 규칙 대로 FORMAT 파라미터 이후에 열거된 파라미터 값으로 차례대로 대치되어 결과 문자열을 생성한다.(Remark 참조)
  %?%

? 자리 문자열 (? 는 숫자임)

오른쪽부터 문자열을 채우고, 나머지는 공백으로 문자열을 채운다. 만약, 파라미터 값이 n 자리보다 크면 n 자리를 넘어서는 문자들은 생략된다.
  %?_*%

? 자리 문자열 (? 는 숫자임, * 는 문자임)

오른쪽부터 문자열을 채우고, 나머지는 *부분에 지정한 문자로 문자열을 채운다. 만약, 파라미터 값이 n 자리보다 크면 n 자리를 넘어서는 문자들은 생략된다.
  %-?%

? 자리 문자열 (? 는 숫자임, * 는 문자임)

왼쪽부터 문자열을 채우고, 나머지는 공백으로 문자열을 채운다. 만약, 파라미터 값이 n 자리보다 크면 n 자리를 넘어서는 문자들은 생략된다.
  %-?_*%

? 자리 문자열 (? 는 숫자임, * 는 문자임)

왼쪽부터 문자열을 채우고, 나머지는 * 부분에 지정한 문자로 문자열을 채운다. 만약, 파라미터 값이 n 자리보다 크면 n 자리를 넘어서는 문자 들은 생략된다.
X1, X2 [in] FORMAT 문자열에 %?% (또는 %-?%) 부분에 차례대로 대치될 값

 

 

Return:

 

STR

결과 문자열

 

 

Remark:

 

ex)

str_format (‘이름=%10%, 나이=%-3%세, 출근일수=%-5_0%일’, NAME_VAR, AGE_VAR, DAYS_VAR)

⇒ ‘이름=홍길동, 나이= 25세, 출근일수=78500일’
