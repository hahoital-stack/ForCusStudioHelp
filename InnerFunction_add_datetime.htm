DATE_TIME add_datetime( DATETIME_FLAG, START_DATE_TIME, NUMBER ) 시작  다음  이전

add_datetime

 

기준 일자 시각에서 특정 시간정보를 더한 결과 일자를 리턴 한다.

 

 

Parameter:

 

DATETIME_FLAG

[in] NUMBER 에 입력되는 정보에 대한 선택 사항. 대소문자를 유념하여 사용 해야 한다.
  'Y'

: 연 수
  'M'

: 월 수
  'D'

: 일 수
  'h'

: 시간
  'm'

: 분
  's'

: 초
START_DATE_TIME [in] 기준일자시각. YYYYMMDDhhmmss
NUMBER [in] 시간정보. DATETIME_FLAG 에 의해 연 수, 월 수, 일 수, 시간, 분, 또는 초 이다. 음수이면, 이전 일자 시각을 얻는데 사용된다.

 

 

Return:

 

DATE_TIME

START_DATE_TIME 에서 DATETIME_FLAG 입력 값에 따라 NUMBER 입력 값만큼 연 수, 월 수, 일 수, 시간, 분, 또는 초 가 더해진 일자 시각을 리턴한다.

NUMBER가 음수이면 이전 일자 시각이 구해진다.

DATETIME_FLAG 값이 올바르지 않으면, 에러의 경우로 아무것도 리턴 하지 않는다.

 

 

Remark:

 

ex)

add_datetime('h', '20130928110710', 5)

⇒ '20130928160710'
