Get Lock

 

- 형상관리 중인 파일에 락을 걸어서 락은 건 사람이 아닌 다른 사용자가 Commit 할 수 없도록 합니다.

 

 

Lock message : 잠금 메시지를 입력 합니다.

 

Steal the locks : 파일이 잠겨 있는 경우 이전 소유자의 잠금을 해제하지 않고 파일을 잠그고 업데이트 해야 하는 경우 사용 합니다.

 

 

Get Locks 후 리스트 Lock, Lock comment 컬럼에 상태가 표시 됩니다.
