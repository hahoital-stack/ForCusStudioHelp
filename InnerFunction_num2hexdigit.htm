num2hexdigit

 

숫자 형식의 문자열을 Hexa String 으로 변환하되 Digit에 전달한 숫자 만큼 앞에 0을 채워 반환.

 

단, 변환된 Hexa String 이 Digit값 보다 큰 경우에는 변환된 Hexa String 그대로 반환한다.

 

∗ ForCus v5.2.0-b3 이상 지원

 

 

Parameter:

 

STRING [in] 입력 숫자 문자열
NUMBER [in] 입력 숫자 Digit

 

 

Return:

 

OUTPUT_STR 결과 Hexa String

 

 

Remark:

 

ex)

num2hexdigit( '10', 2 )

⇒ '0A'

 

num2hexdigit( '16', 2 )

⇒ '1A'

 

num2hexdigit( '16', 4 )

⇒ '001A'
