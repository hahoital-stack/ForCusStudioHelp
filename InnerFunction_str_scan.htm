str_scan

 

입력 문자열의 내용을 입력된 포멧으로 파싱합니다.

제공하는 FORMAT 에 한정하여 데이터를 파싱합니다.

str_scan은 파싱 후 남은 문자열을 리턴하기 때문에 남은 문자열에 대한 파싱이 가능하므로 같은 크기의 문자(문자열) 파싱을 하는 경우 연속으로 호출하여 처리가 가능하다.

 

 

Parameter:

 

STRING

[in] 문자열 데이터
FORMAT

[in] 파싱 포멧

ForCus에서 제공되는 포멧은 C언어에서 사용되는 일반적인 포멧은 아니다.

%d, %s, %c, %o, %x 등의 포멧은 사용하지 않고 기본은 %s(문자열) 기능이다.
  %?

? 자리 문자열 (? 는 숫자), ? 자리 숫자만큼 이동하면서 문자열 파싱

설정한 문자열 길이가 전체 문자열 길이보다 길면 전체 문자열을 가져온다.
  *

문자열 데이터에서 * 위치에 해당하는 문자(문자열)

*위치에 해당하는 문자(문자열)가 없으면 파싱 중단하고 NULL 리턴
X1, X2

[out] 문자열 파싱 FORMAT 에 차례대로 대치될 출력 값

실패면 출력값 NULL

 

 

Return:

 

STR

파싱 후 남은 문자열

실패이거나, 남은 문자열이 없는 경우 NULL

 

 

Remark:

 

ex)

test_str = '테스트 1:data 1+2+3=6 입니다'
변수 : var1, var2, var3, var4, var5, var6
str_scan(test_str, '%8:%4 %5%1%1 %6', var1, var2, var3, var4, var5, var6)
⇒ var1 : '테스트 1'
     var2 : 'data'
     var3 : '1+2+3'
     var4 : '='
     var5 : '6'
     var6 : '입니다'


* 일정크기 문자열로 파싱
test_str = str_scan(test_str, '%2%4', var1, var2)
⇒ var1 : '테'
     var2 : '스트'
     test_str : ' 1:data 1+2+3=6 입니다'
