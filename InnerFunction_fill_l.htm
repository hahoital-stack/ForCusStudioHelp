OUTPUT_STR fill_l( STRING, FULL_LEN, FILL_CHAR ) 시작  다음  이전

fill_l

 

입력문자열 앞에 특정 문자로 채워진 지정된 길이의 문자열을 리턴 한다.

 

 

Parameter:

 

STRING [in] 입력 문자열
FULL_LEN [in] 결과 문자열의 길이
FILL_CHAR [in] 채울 문자

 

 

Return:

 

OUTPUT_STR

결과 문자열

채우기 실패 시, 입력 문자열을 리턴 한다.

 

 

Remark:

 

fill_l() 함수는 STRING 문자열이 FULL_LEN 길이가 되도록, 문자열 앞에 FILL_CHAR 문자로 채운 문자열을 리턴 한다. 만약, FULL_LEN 입력 값이 실제 입력문자열의 길이보다 작거나 같을 때, 입력문자열을 그대로 리턴 한다. 만약, FULL_LEN 이 숫자 값이 아니면, 입력문자열을 그대로 리턴 한다.

 

ex)

fill_l( '_Sunday', 10, 'a' )

⇒ 'aaa_Sunday'

 

fill_l( '_Sunday ', 10, 'abc' )

⇒ 'aaa_Sunday' : 문자열 'abc'를 입력해도 가장 앞의 문자 'a'로 채워진다

 

fill_l( '', 10, 'a' )

⇒ 'aaaaaaaaaa' : 입력 문자열이 NULL인 경우에도 자리수 만큼 채운다

 

fill_l( 'Sunday', 5, 'a')

⇒ 'Sunday' : 결과 문자열의 길이가 결과보다 작아서 채우기 실패
