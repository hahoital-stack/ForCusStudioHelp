■ ASCII 입력

 

ASCII ( American Standard Code for Information Interchange ) 는 미국 국립 표준 협회 ( American National Standards Institute, ANSI ) 에서 표준화한 정보교환용 7비트 부호체계이다.

 

ForCus 시나리오에서 ASCII 를 사용해야 하는 대표적인 경우는 문자열 입력에서 "(쌍따옴표) 입력 시 인데 SXML 데이터 표현이 Key="Value" 구조로 하기 때문에 "(쌍따옴표)를 바로 입력하면 태그오류가 발생하기 때문에 할 수 없다.

그외에 시나리오 데이터가 SXML 구성과 중복되는 경우 ASCII 로 입력 해야 한다.

 

사용방법

 

표현 : 역슬래쉬x아스키

※ 아스키는 Hex 값으로 입력해야 하며 영문이 포함된 경우 대문자로 해야한다.

 

예제 : 문자열 '안녕하세요? 김영희 입니다.' 에서 '김영희' 앞뒤로 "(쌍따옴표) 입력

'안녕하세요? \x22김영희\x22 입니다.'
