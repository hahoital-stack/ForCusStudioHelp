Working copy browser 기능

 

 

 

Working copy browser 각 항목에서 마우스 오른 버튼을 누르면 항목의 상태에 따라서 SVN 메뉴가 팝업 됩니다.

 

Working copy browser 에 있는 SVN 기능은 전체 솔루션 항목이 대상이고 각 항목에서 팝업메뉴의 SVN 기능은 개별 항목에 대한 처리를 한다.

 

아래 명령에서 (SVN 명령 아님)기능은 윈도우 탐색기와 같이 실제 파일 처리 명령입니다. 형상관리 중인 파일에 대해서는 Subversion 기능으로 처리해야 관리에 문제가 없습니다.

 

Subversion : SVN 명령

-------------------

Cut (Ctrl+X) : 실제 파일 잘라내기(SVN 명령 아님)

Copy (Ctrl+C) : 실제 파일 복사(SVN 명령 아님)

--------------------

Delete (Del.) : 실제 파일 삭제(SVN 명령 아님), 단 형상관리 중인 항목은 SVN Delete를 합니다.

Rename (F2) : 실제 파일 이름 변경(SVN 명령 아님), 단, 형상관리 중인 항목은 SVN Rename을 합니다.
