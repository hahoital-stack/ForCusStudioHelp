Check-modifi...

 

- 수정, 추가, 삭제된 파일들을 리스팅 합니다.

 

 

List : Commit 하기 전의 수정된 파일 또는 추가, 삭제 된 파일 리스트 입니다.

 

Directories Checkbox : 디렉토리를 리스팅 합니다.

Files Checkbox : 파일을 리스팅 합니다.

 

NotVersioned Checkbox : 형상관리 중이 아닌 파일 리스팅 합니다.

Normal Checkbox : 형상관리 중인 보통 파일을 리스팅 합니다.

Ignore Checkbox : 형상관리 제외 파일을 리스팅 합니다.

 

Revert Popup Menu : 수정, 추가, 삭제된 파일을 이전 상태로 되돌릴 수 있습니다.

Added 된 항목을 Revert 하면 이전 상태인 NotVersioned 또는 Ignored 상태로 돌아 가고 리스트에서 제거 됩니다.
