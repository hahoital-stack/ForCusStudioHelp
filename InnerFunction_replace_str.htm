replace_str

 

문자열 내에 특정 문자열을 다른 문자열로 대체한 결과 문자열을 리턴 한다.

 

 

Parameter:

 

STRING [in] 입력 문자열
SUB_STR [in] 교체 대상의 문자열
REPLACE_STR [in] 대체 문자열

 

 

Return:

 

OUTPUT_STR

결과 문자열

치환 실패 시, 입력 문자열을 리턴 한다.

 

 

Remark:

 

REPLACE_STR 에 비어있는 문자열을 입력하면, 입력 문자열에서 교체 대상 문자열을 지운 결과 문자열을 리턴 한다. erase_str() 함수와 동일한 기능을 수행한다.

 

ex)

replace_str( 'Refreshing Sunday morning', 'Sunday', 'Monday' )

⇒ 'Refreshing Monday morning'

 

replace_str( 'Refreshing Sunday morning', 'Saturday', 'Monday' )

⇒ 'Refreshing Sunday morning' : 교체 대상 문자열이 없어서 치환 실패
