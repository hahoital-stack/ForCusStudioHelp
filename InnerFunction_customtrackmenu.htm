customtrackmenu

 

해당 메뉴에 진입하지 않고 메뉴 인입 카운트를 증가 시켜 준다.

 

ForCus v4.x 에서 제공하던 _CDR_MENULIST_ 를 수정할 수 있는 기능을 v5.0 부터는 제공하지 않아서 기존에 사용하던 시나리오를 컨버팅하여 v5.x에서 사용하려 할 때 영향도를 최소화 할 수 있도록 내장함수로 제공한다.

 

Parameter:

 

STRING [in] 인입 카운트를 증가 할 대상의 Menu ID

 

 

Return:

 

INT_NUMBER 해당 메뉴에 전달된 Count값이 더해진 값

 

 

Remark:

 

ex)

addmenucnt('Menu2')

⇒ Return Value 3
