DATE add_date ( DATE_FLAG, START_DATE, NUMBER ) 시작  다음  이전

add_date

 

기준 일자에서 특정 시간정보를 더한 결과 일자를 리턴 한다.

 

 

Parameter:

 

DATE_FLAG

[in] NUMBER 에 입력되는 정보에 대한 선택 사항. 대소문자를 유념하여 사용 해야 한다.
  'Y'

: 연 수
  'M'

: 월 수
  'D'

: 일 수
START_DATE [in] 기준일자. YYYYMMDD
NUMBER [in] 시간정보. DATE_FLAG 에 의해 연 수, 월 수, 또는 일 수 이다. 음수이면, 이전 일자를 얻는데 사용된다

 

 

Return:

 

DATE

START_DATE 일자에서 DATE_FLAG 입력 값에 따라 NUMBER 입력 값만큼 연 수, 월 수, 또는 일 수가 더해진 일자를 리턴 한다.

NUMBER 가 음수이면 이전 일자가 구해진다.

DATE_FLAG 값이 올바르지 않으면, 에러의 경우로 아무것도 리턴 하지 않는다.

 

 

Remark:

 

ex)

add_date('D', '20130928', 2)

⇒ '20130930'
