Checkout

 

- 선택한 디렉토리를 로컬에 SVN 정보를 포함하여 다운로드 합니다.

+ Checkout 받은 디렉토리에 .svn 이라는 디렉토리가 생성되는데 여기에 SVN 형상관리 정보가 모두 들어있으므로 이 폴더를 삭제하면 형상관리가 안됩니다.

 

 

URL of remote repository : Checkout 할 Repository URL 주소를 입력 합니다.(솔루션 시나리오를 모두 받을 때는 /trunk 에서 체크 아웃 합니다.)

Checkout directory : Checkout 받을 로컬 디렉토리를 입력(선택) 합니다.(기존에 Checkout 받지 않은 빈 디렉토리를 선택 하세요)

 

Revision : Checkout 할 Revision을 선택합니다.

 

 

빈 디렉토리가 아닌 경우 경고 창이 팝업 됩니다.

 

 

HEAD Revison:144 이고 4286개 가 Checkout 되었습니다.

 

 

.svn 디렉토리와 솔루션에 소속된 디렉토리 및 파일이 Checkout 되었습니다.
