GotoLabel 시작  다음  이전

시나리오 영역(시나리오 파일 단위) 내에서 지정된 위치로 이동합니다.(SetLabel Item) 긴 링크를 대신 시나리오 작성을 좀더 쉽게 할 수 있습니다.

 

 

ICON

 

 

 

PROPERTIES

 

Label Info

 

 

Tag Name : SetLabel 아이템의 List of GotoLabel Items 리스트에서 각 GotoLabel 아이템을 구분해 주는 이름입니다.

 

Label ID : 이동할 라벨 아이디를 선택 합니다.(SetLabel 의 Label ID 속성)

 

Goto Item Button : 지정된 라벨(SetLabel Item)로 이동 합니다.(설정되어 있는 SetLabel 을 지정해야 Enable됩니다)

 

    편집 화면에서 GotoLabel 아이템을 더블클릭 하면 지정된 SetLabel 아이템으로 이동 합니다.

 

Label Desc : SetLabel 에 지정한 라벨의 설명이 표시 됩니다.

 

 

RESULT

 

None.
