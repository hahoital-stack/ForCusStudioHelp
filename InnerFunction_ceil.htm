ceil

 

부동소수를 올림 계산한 정수 값을 리턴 한다.

 

 

Parameter:

 

FLOAT_NUMBER [in] 입력 부동 소수

 

 

Return:

 

INT_NUMBER

성공일 경우 정수

실패일 경우 아무 것도 리턴 하지 않는다.

 

 

Remark:

 

FLOAT_NUMBER 가 정상적인 부동소수가 아닐 경우에는 연산이 실패된다. 예를 들어, ‘a’, ‘3.2.2’ 와 같은 문자열이 입력되면 연산이 실패되고 아무 것도 리턴하지 않는다.

 

입력 값이 음수일 경우 Round Down(0에 가까워지도록 수를 내림)으로 동작 합니다.

 

ceil 함수의 실행 결과는 다음과 같다.

 

ex) ceil(2.8) ＝ 3
  ceil(2.3) ＝ 3
  ceil(-2.8) ＝ -2
  ceil(-2.3) ＝ -2
