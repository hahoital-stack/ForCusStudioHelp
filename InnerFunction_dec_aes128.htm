dec_aes128

 

AES128로 데이터를 복호화 합니다.(BASE-64 Decoding)

 

* ECB 방식이고 NULL Padding 처리 합니다.

 

 

Parameter:

 

STRING

[in] 대칭키

입력이 NULL이면 내부 키 이용(SWAT, SLEE간)
STRING [in] 암호화된 문자열(BASE-64 인코딩)

 

 

Return:

 

OUTPUT_STR

복호화된 문자열(평문 데이터)

복호화 실패 시 NULL

 

 

Remark:

 

ex)

dec_aes128('aestest_key123!@@#$', '3690cb08c9b2cbcc0e9caf60e5cad38dc' )

⇒ '테스트 데이터'
