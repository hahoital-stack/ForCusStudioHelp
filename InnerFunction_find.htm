find

 

문자열 내에서 지정된 문자열을 찾는다.

 

 

Parameter:

 

STRING [in] 입력 문자열
FIND_STR [in] 찾을 문자열
START_POS [in] 입력 문자열중, 검색 시작 위치 (1-base)

 

 

Return:

 

POS

입력 문자열내의 찾을 문자열(FIND_STR 입력 값)의 위치(1-base).

만약, 찾을 문자열이 입력 문자열 내에 없다면, 0 을 리턴한다.

 

 

Remark:

 

START_POS 입력 값이 1 보다 작거나 입력 문자열의 길이보다 길다면, 검색작업을 수행하지 않고 0을 리턴 한다.

 

ex)

find( 'Refreshing Sunday morning', 'morning', 1)

⇒ 19

 

find( 'Refreshing Sunday morning', 'lunch', 1)

⇒ 0
