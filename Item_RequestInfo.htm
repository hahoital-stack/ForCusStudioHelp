RequestInfo 시작  다음  이전

서비스 메뉴 코드, 앱 설치 및 활성화 여부를 요청 하는 아이템 입니다.

 

 

ICON

 

 

 

PROPERTIES

 

Mobile Info

 

 

Name : 아이템 이름을 입력합니다.(결과 시스템 변수를 참조할 때 사용 합니다)

 

Server Type : TB 에서 전문을 전송할 대상 서버의 타입을 선택합니다.(MTM(default), ICM)

 

Transaction ID : TB 에서 전문을 전송할 대상(iWeb-MTM Adapter, iWeb-ICM Adapter)을 찾기 위한 서버 키(SWAT HOME>구성관리>IVR 서비스 관리>확장 어댑터 관리(CB/TB) : Trans ID) 입니다. Server Type이 ICM 일 경우 필수 입력 사항입니다.

 

Type : 요청할 정보를 선택 합니다.

Service Information : 앱 서비스 정보 요청

Service Code : 서비스 메뉴 코드 요청

Active Application : 앱 활성화 여부 요청

Active Session : 세션(ICM) 활성화 여부 요청

 

TelNo : 정보를 요청할 전화번호를 설정합니다.

 

Condition : 아이템 수행 조건을 지정 합니다.

 

 

RESULT

 

· 처리 결과 세션 변수

 

session.command.result : 처리 성공(_SLEE_SUCCESS_), 실패(_SLEE_FAILED_)

session.command.reason : 실패면 실패 오류코드(시스템 상수 참조)

 

· 아이템 속성 변수(@name 은 Name 항목에서 지정한 이름)

 

@name.svccode : 서비스 메뉴 코드

@name.activeapp : 앱 설치 및 활성화 여부(1:Active, 0:Unactive)

@name.ostype : OS Type((0:iPhone, 1:Android, 9:Unknown)

@name.activesession : 세션(ICM) 활성화 여부(1:Active, 0:Unactive)
